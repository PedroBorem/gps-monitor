GPS Monitor preparado para publicação

Esta página pode ser publicada na Vercel sem expor certificados. O processo MQTT da AWS IoT Core continua em monitor.py, rodando localmente ou em uma VM, porque a assinatura contínua de tópico não combina bem com funções serverless.

O que sobe para a Vercel

  • Esta interface estática em public/index.html.
  • Documentação do fluxo de monitoramento.
  • Nenhum certificado ou arquivo sensível.

O que fica fora da Vercel

  • KEYS/ com os certificados.
  • config.json com endpoint local.
  • O processo Python que publica em PIVO_ID e assina cloudv2-GPS.

Execução local do monitor

pip install -r requirements.txt
python monitor.py --serve

Se você quiser, o próximo passo depois do deploy é adaptar essa interface para consumir uma API externa ou armazenamento remoto com os dados produzidos pelo monitor.

O repositório também ficou pronto para importar na Vercel pelo GitHub.
